Total Commander Edit Redirector, Версия 1.0.0.0
    Утилита для Total Commander, позволяющая открывать файлы на редактирование в разных редакторах,
    в зависимости от расширений.

Автор утилиты:     Власов Константин, 2010 г.
Домашняя страница: http://flint-inc.ru/
E-mail:            support@flint_inc.ru

Описание
--------

Использование редактора файлов в Total Commander реализовано с ограничениями, а именно:
а) невозможно назначить разным типам файлов разные редакторы;
б) невозможно отправить в редактор сразу несколько файлов.
Для решения этих двух проблем и была разработана программа TCER.


Установка
---------

Для установки программы скопируйте её в любой каталог. Файл tcer.ini нужно поместить либо рядом с
исполняемым файлом программы, либо (если этот каталог недоступен для записи) в каталог, где
находится основной файл конфигурации TC (wincmd.ini).
После этого откройте диалог настройки TC на странице "Правка/Просмотр" и укажите полный путь к файлу
tcer.exe в поле "Программа-редактор".
Настройка программы осуществляется ручной правкой INI-файла.


Файл конфигурации
-----------------
[Configuration]         Секция общих настроек программы.
MaxItems=256            При превышении указанного здесь количества выделенных файлов будет выведен
                        дополнительный запрос-подтверждение.
ClearSelection=1        Если значение 1, в окне TC выделение будет автоматически снято после запуска
                        редактора.

[Extensions]            Секция привязки расширений к программам.
ext=Editor              ext - расширение файла (<nil> - отсутствующее); программа-редактор
                        определяется в секции [Program_Editor].

[DefaultProgram]        Параметры редактора, вызывающегося по умолчанию.
[Program_Editor1]       Параметры программы Editor1.
FullPath=%ProgramFiles%\MyEditor\MyEditor.exe
                        Полный путь к исполняемому файлу редактора. Можно использовать переменные
                        окружения.
MDI=1                   1, если редактор многодокументный; 0, если однодокументный.


Ограничения программы
---------------------

1. В режиме эскизов открывается только файл под курсором (ограничение TC).
2. Каталоги %TEMP%\_tc и %TEMP%\_tc_ считаются служебными, в них открывается только файл под
   курсором (временное решение, возможна доработка в будущем).
3. Не поддерживаются виртуальные папки (возможна доработка в будущем).
4. В архивах, FS-плагинах, при FTP/LPT/USB-подключениях будет открыт только файл под курсором
   (ограничение TC).
5. Если выбранные файлы привязаны к разным редакторам, они все откроются в одном редакторе - первом
   (возможна доработка в будущем).


ЧаВо (ЧАсто задаваемые ВОпросы)
-------------------------------

В: Зачем ещё один мультиредактор? Есть же ChoiceEditor, OFSTC, F4Menu.
О: Мне захотелось написать такую программу для себя, чтобы я мог не зависеть от авторов этих
   программ, чтобы мог сам исправлять ошибки и дорабатывать утилиту под свои нужды. По
   функциональности TCER никогда не превзойдёт упомянутые альтернативы, такой цели не ставилось.
   Основной упор делается на минимальный необходимый функционал, максимальную компактность и
   скорость работы. Если кому-то программа будет полезной - хорошо, нет - можно выбрать любую
   другую. В конце концов, много - не мало.

В: Не хочу копаться в INI-файле! Почему нет удобного конфигуратора?
О: В целях минималистичности. Не исключено, что конфигуратор будет написан и включён в пакет в виде
   отдельной утилиты, но эта задача для меня низкоприоритетна. Даже если конфигуратор будет
   разработан, интеграции его с TCER не планируется (опять же, по соображениям минималистичности).

В: А как назначить несколько программ на одно расширение с возможностью выбора?
О: Никак. Эта функция не поддерживается и не планируется.
